# 🛢️ Классификатор смазочных материалов

**Интеграция ИКПУ • ЕНКТ • ТН ВЭД • СКП**

Публичный объединённый справочник масел, пластичных смазок и охлаждающих
жидкостей: 561 позиция, 22 бренда/источника и 25 категорий. Включает новый
реестр CHILON/UNO, прежнюю мультибрендовую базу и отраслевые классы смазок.

## 🚀 Демо

🔗 **[Открыть классификатор](https://temuradhamson.github.io/mfclassifier/)**

## 📋 Возможности

### Фильтрация и поиск
- 🔍 Быстрый поиск по названию, коду, спецификации
- 🏭 Фильтр по бренду
- ⚙️ Фильтр по классу вязкости SAE
- 📋 Фильтр по стандартам API/ACEA/ГОСТ
- 📦 Фильтр по упаковке
- 🧪 Фильтр "Для теста"

### Просмотр данных
- 📊 Табличный вид с сортировкой
- 🃏 Карточный вид для быстрого просмотра
- 📈 Статистика и диаграммы
- 🔎 Детальная информация по каждому продукту

### Экспорт
- 📄 CSV (для Excel)
- 📋 JSON (для API)
- 🖨️ Печать
- 📎 Копирование в буфер

## 📊 Поля данных

| Поле | Описание |
|------|----------|
| **Наименование** | Полное название продукта |
| **Класс вязкости** | SAE (0W-20, 5W-30, 10W-40 и др.) |
| **Стандарт** | API (SN, CF, CJ-4 и др.) |
| **Спецификация** | ACEA/ГОСТ |
| **Ед.изм** | Единица измерения |
| **Упаковка** | Тип упаковки (канистра, бочка) |
| **Бренд** | Производитель |
| **Тара** | Базовая ед.изм + упаковка |
| **Для теста** | Флаг тестовой позиции |
| **ИКПУ** | Код налогового классификатора |
| **ЕНКТ** | Код классификатора МинФин |
| **ТН ВЭД** | Код товарной номенклатуры ВЭД |
| **СКП** | Статистический классификатор продукции |

## 🏷️ Классификаторы

### ТН ВЭД (Глава 27)
- `2710 19 82` - Моторные масла, компрессорные, турбинные
- `2710 19 83` - Гидравлические масла
- `2710 19 84` - Гидравлические жидкости

### ИКПУ / ЕНКТ
- `19.20.29.210` - Масла смазочные

## 📚 Источники данных

- **Каталог продукции АЗМОЛ** - технические характеристики масел
- **ТН ВЭД ЕАЭС** - коды товарной номенклатуры
- **ГОСТ 17479.1-2015** - классификация моторных масел
- **SAE J300** - классификация по вязкости
- **API 1509** - спецификации качества

## 🌐 Стандарты

| Стандарт | Назначение |
|----------|------------|
| **SAE** | Вязкость (Society of Automotive Engineers) |
| **API** | Качество (American Petroleum Institute) |
| **ACEA** | Европейская классификация |
| **ГОСТ** | Государственный стандарт |
| **ISO** | Международная стандартизация |
